搭建个人博客

准备工具

Node.js安装

安装教程：https://www.cnblogs.com/zhouyu2017/p/6485265.html

Git安装

我在官网下载得太慢，又找一个下载链接：https://webcdn.m.qq.com/spcmgr/download/Git-2.38.1-64-bit.exe

Hexo下载

cmd指令 npm install express -g

搭建Github仓库

打开github新建仓库，注意仓库名为用户名

添加readme文件，然后创建仓库

生成ssh key

打开hexo bash here生成ssh:ssh-keygen -t rsa -C “github注册的邮箱地址”

.ssh生成路径：C:\Users\用户名.ssh
用记事本打开ssh文档中的id_rsa.pub，复制里面的全部内容，打开github的setting，然后创建新的ssh key，标题随意，将复制的内容粘贴到key里面，点击add

测定ssh是否绑定成功：ssh -T git@github.com

如果一直出现ssh: connect to host github.com port 22: Connection refused的话具体看这个教程 https://www.cnblogs.com/Archer314/p/14641310.html

生成本地博客内容

新建文件夹，在文件夹里打开hexo bash here
hexo init #初始化hexo博客

如果报错SyntaxError: Unexpected token …

建议更新一下node.js版本

hexo s #静态生成hexo页面

博客发布到互联网

打开之前新建的文件，在显示中打开文件扩展名，打开_config.yml，在最后添加
deploy:

type: git

repository:https://github.com/你的名字/仓库的名字.git

branch: 分支
安装hexo-deployer-git自动部署发布工具:
npm install hexo-deployer-git –save

hexo g #生成页面
hexo d #上传
第一次上传需要绑定github账号，按照提示绑定即可。
在github的仓库设置中打开pages可以查看自己博客的网址
